# 2D Graphics Editor in C

This project is a menu-driven 2D graphics editor developed using C programming language and a 2D character array.

## Features
- Draw Rectangle
- Draw Horizontal Line
- Draw Vertical Line
- Draw Triangle
- Draw Circle
- Delete Area
- Modify Rectangle
- Display Graphics Board

## Technologies Used
- C Programming
- 2D Arrays

## Description
The project allows users to create and manage simple text-based graphics using '*' and '_' characters.
