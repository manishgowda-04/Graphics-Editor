# 2D Interactive Graphics Editor in C

This project is a menu-driven interactive 2D graphics editor developed using C programming language and a 2D character array.

## Features
- Interactive cursor movement
- Real-time drawing system
- Rectangle drawing tool
- Horizontal line tool
- Vertical line tool
- Triangle drawing tool
- Circle drawing tool
- Erase functionality
- Clear board option
- Keyboard-based controls
- Display graphics board

## Technologies Used
- C Programming
- 2D Character Arrays
- Windows Console Functions

## Controls
- Arrow Keys / WASD : Move Cursor
- Space / Enter : Draw or Select
- Number Keys (1-7) : Select Tools
- C : Clear Board
- Q / ESC : Exit Program

## Platform
This project is designed for Windows Console using:
- windows.h
- conio.h

## Description
The editor allows users to create and manage simple text-based graphics using '*' and '_' characters in an interactive console environment.
